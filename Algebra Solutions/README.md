# Algebra Solutions

This directory contains detailed solutions to selected Olympiad algebra problems written by **Aryan Goutam**.

The solutions emphasize mathematical rigor, clarity of exposition, problem-solving techniques, and elegant arguments. Problems are drawn from various national and international mathematical olympiads, including RMO, INMO, AMC, AIME, and other prestigious competitions.

Each solution aims not only to establish the result but also to highlight the key ideas, motivations, and methods that make the problem instructive.

## Contents

* Olympiad Algebra Problems
* Functional Equations
* Polynomial Problems
* Inequalities
* Algebraic Number Theory Applications
* Miscellaneous Advanced Topics
