# Polynomial Problems

This collection contains original and classical polynomial problems ranging from elementary identities to Olympiad-level functional and algebraic techniques.

Topics include:

* Polynomial identities
* Root structures and factorization
* Vieta's relations
* Functional equations involving polynomials
* Interpolation and reconstruction
* Symmetric and cyclic polynomial methods
* Advanced Olympiad techniques

The problems are intended to develop creativity, algebraic insight, and problem-solving ability beyond standard textbook exercises.
