# Beginner Olympiad Algebra

A collection of introductory algebra problems and solutions from AMC 8, AMC 10, IOQM, NMTC, AoPS, and other entry-level mathematical olympiads.

The solutions focus on developing algebraic intuition, proof-writing skills, and core problem-solving techniques essential for higher-level Olympiad mathematics.

**Author:** Aryan Goutam
**Drayan Mathematical Series**
