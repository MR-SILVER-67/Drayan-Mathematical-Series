
# Geometry Solutions

A well-organized collection of **geometry problem solutions** with clear explanations, diagrams (where possible), and rigorous proofs.

## 📚 About This Repository

This repository contains step-by-step solutions to various geometry problems, covering topics such as:

- Circle geometry (tangency, intersection, angles, power of a point, etc.)
- Triangle theorems and properties
- Coordinate geometry
- Transformations and symmetry
- 3D geometry
- Olympiad-level geometry problems
