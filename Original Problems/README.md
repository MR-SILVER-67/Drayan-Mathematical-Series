# Original Olympiad Problems (Drayan Mathematical Series)

This folder contains a curated collection of original Olympiad-level problems created as part of the Drayan Mathematical Series.

---

## Purpose

This section demonstrates problem creation ability at competition level, including:

- Non-standard problem design  
- Deep use of classical and modern techniques  
- Exploration of geometry, number theory, algebra, and combinatorics  
- Problems inspired by IMO, RMO, ISL, and national Olympiads, but independently constructed  
- Emphasis on elegance, difficulty balance, and originality  

---

## Nature of Problems

Each problem is crafted to reflect:

- Olympiad-style depth (RMO–IMO range)  
- Clean and precise statement design  
- Hidden structure requiring insight rather than computation  
- Potential for multiple solution approaches  

---

## Goal of This Folder

This is not just practice material.

It is a portfolio of mathematical creativity.

It directly answers:

> Can the author design Olympiad-quality problems, not just solve them?

---

## Structure

Each original problem (when added) follows:

- Problem statement (clean LaTeX format)  
- Difficulty rating (self-assessed)  
- Source tag: `Original`  
- Solution (optional / separate file if included later)  
- Remarks or key idea (optional)  

---

## Scope

Problems span:

- Euclidean Geometry  
- Number Theory  
- Algebra  
- Combinatorics  
- Functional Equations (occasionally)
